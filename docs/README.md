# Elite Tax & Finance Website

A clean, professional website for Elite Tax & Finance business, featuring modern design and responsive layout.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Contact Form**: Interactive contact form with validation
- **Service Showcase**: Highlighting key tax and financial services
- **About Section**: Placeholder for experience and credentials
- **Mobile Navigation**: Hamburger menu for mobile devices

## Sections

1. **Hero Section**: Eye-catching landing area with call-to-action buttons
2. **About Section**: Business information with placeholder for detailed experience
3. **Services Section**: Four key service offerings with icons
4. **Contact Section**: Contact information and functional contact form
5. **Footer**: Additional links and contact details

## Getting Started

### Prerequisites

- Node.js (optional, for development server)
- Modern web browser

### Installation

1. Clone or download the website files
2. Navigate to the website directory:
   ```bash
   cd website
   ```

### Running the Website

#### Option 1: Simple File Opening
- Open `index.html` directly in your web browser

#### Option 2: Local Development Server
1. Install dependencies (optional):
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   or
   ```bash
   npm start
   ```

3. Open your browser to `http://localhost:3000`

## Customization

### Contact Information
Update the contact details in `index.html`:
- Phone number
- Email address
- Physical address
- Business hours

### Business Information
- Update the business name throughout the files
- Add your actual experience and credentials in the About section
- Modify service descriptions to match your offerings

### Styling
- Colors and fonts can be customized in `styles.css`
- The main brand color is currently set to `#2563eb` (blue)
- Gradient colors can be adjusted in the hero section

### Form Handling
The contact form currently shows a success message. To make it functional:
1. Set up a form handling service (Formspree, Netlify Forms, etc.)
2. Update the form action in `index.html`
3. Modify the JavaScript in `script.js` to handle real form submission
